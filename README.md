
# AI Car

A deep q learning approach for self driving 2d car with P5js and tensorflow.js


## Run Locally

Clone the project

```bash
  git clone https://github.com/ars2062/AICar
```

Go to the project directory

```bash
  cd AICar
```

Install dependencies

```bash
  npm install
```

Build the project

```bash
  npm run build
```

Start the server

```bash
  npm run start
```

for development you can run

```bash
  npm run dev
```
